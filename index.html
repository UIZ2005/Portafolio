Portafolio

SafeRoute

SafeRoute es una aplicación web que permite clasificar el estado de una calle (Calle Buena, Grietas o Huecos) mediante una red neuronal CNN y detectar anomalías con un modelo YOLO. Además, extrae la ubicación GPS desde los metadatos EXIF de la fotografía y la muestra en un mapa interactivo con Leaflet.

Tecnologías:

Python TensorFlow, YOLOv8, OpenCV HTML, CSS, JavaScript Flask

BURN FIGHT

Desarrollo de un videojuego promocional (advergame) diseñado para fortalecer la presencia de marca de un protector solar en eventos presenciales. El juego, de corta duración y dinámica rápida, fue optimizado para pantallas táctiles y entornos de stand interactivo. Además, implementa un sistema de registro y almacenamiento de puntajes en formato JSON para identificar a los mejores jugadores al final de la jornada.
